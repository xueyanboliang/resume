# 我的PC端简历

> 闲来无事在网上浏览网页时发现一个设计特别棒的关于饮料的网站，突发奇想想借鉴它的创意做一个自己的PC端简历，  <br/>
好让自己的简历看起来更阳光，帅气和“迷人”一点，于是话不多说便用原生的JS写了一个,希望大家多多指教。

## 技术栈
+ 原生JS。

## 功能
+ [x] 吸顶导航
+ [x] 云朵循环动画
+ [x] 页面加载一次性动画
+ [x] 无缝轮播图

## 安装运行
> clone 项目到本地，用浏览器打开index.html即可运行
